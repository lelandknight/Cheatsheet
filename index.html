Color Reference for CSS

Coloring Styles

Color Format Layout Meaning
Hexadecimal #000000 to #ffffff

Hexadecimal colors are written with a # first then a 6 digit code. The codes range in numbers from "0-9" and letters "a-f" to create various colors.

HSL hsl(360, 100%, 100%) HSL stands for Hue, Saturation, and Lightness. The Hue ranges from 0-360 to change the color. The Saturation percentage changes how intense the color appears. The Lightness changes how light or dark the color appears.
RGB rgb(255, 255, 255) RGB stands for Red, Green, and Blue. The numbers range from 0-255 and changing these numbers accordingly changes the color.
Name Keywords Word Format like "Aqua" There are 148 named colors that can be used in CSS. This may be easy to browse and pick a color but it truly limits you on the total number of colors available.
Alpha Values rgba(255, 255, 255, 0.5) or hsla(360, 100%, 100%, 0.5) The Alpha Values can be used in RGB and HSL color formats. The Alpha Value ranges from 0 to 1 and is typically used in 0.1 increments. This value changes the transparency of the color to allow the colors behind it to show through.
